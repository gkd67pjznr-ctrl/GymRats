## Top Recommendations

### 1. Recraft AI (Best for UI/Icons) ‚≠ê
- **Vector-first design** - exports directly to SVG
- Clean, consistent style perfect for app icons and UI elements
- Free tier available with generous limits
- Great for: icons, illustrations, UI components
- [recraft.ai](https://recraft.ai)

### 2. Icons8 Illustration Generator
- Specifically designed for UI illustrations
- Consistent style across generated assets
- Integrates with their icon library
- Free tier with attribution
- [icons8.com/illustrations/ai](https://icons8.com/illustrations/ai)

### 3. Leonardo AI
- Excellent for stylized game/app artwork
- Strong consistency tools for batch generation
- Good free tier (150 daily tokens)
- Great for: character art, backgrounds, effects
- [leonardo.ai](https://leonardo.ai)

### 4. Canva Magic Media
- Easy to use, integrated with design workflow
- Good for quick mockups and social assets
- Free tier included with Canva
- [canva.com](https://canva.com)

### 5. Ideogram (Best for Text in Images)
- Excellent text rendering in images
- Good for badges, notifications, banners
- Free tier available
- [ideogram.ai](https://ideogram.ai)

## Quick Workflow for GymRats

For consistent UI assets in GymRats's "Pure" dark aesthetic:

1. **Recraft AI** for vector icons and UI elements (exports SVG)
2. **Leonardo AI** for character/avatar artwork with style locking
3. **Khroma.co** for AI-generated color palettes that match toxic/electric/ember themes
