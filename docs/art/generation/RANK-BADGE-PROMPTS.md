## Badge Style Guide

- **Size:** 512x512 pixels (DALL-E outputs 1024x1024)
- **Style:** Clean vector, flat lighting, brushed metal texture
- **Background:** Transparent
- **Icon:** Stylized flexed arm holding a dumbbell (2 weight plates per side)
- **Progression (I → II → III):**
  - Level I: Base design with faint inner glow
  - Level II: More intense inner glow and radial highlight
  - Level III: Powerful radiant glow with bold radial highlight
